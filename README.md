# Chill REST API

REST API for the [chill](https://github.com/leapfrogtechnology/chill) monitoring service.

## Setup

Install dependencies
```bash
# Using npm
$ npm install

# Or using yarn
$ yarn
```

Then start the app
```
$ npm start
```
